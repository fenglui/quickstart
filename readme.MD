## 测试环境

### node + wepy + 微信小程序开发工具

nodejs 64位 -[https://nodejs.org/en/]

安装cnpm （npm淘宝镜像）

```console
npm install -g cnpm --registry=https://registry.npm.taobao.org
```

### wepy

全局安装或更新 wepy-cli

```console
cnpm install -g wepy-cli
```
